Linux 中的 Super 键即键盘上带有 Windows 图标的键

​                  

|       super        |  打开活动搜索页面  |
| :----------------: | :----------------: |
|     ctrl+alt+t     |      打开终端      |
|      super+d       |      显示桌面      |
|      super+a       |  显示应用程序菜单  |
|     super+tab      | 在应用程序之间切换 |
| super+上下左右箭头 |   移动窗口的位置   |
|      super+m       |    切换到通知栏    |
